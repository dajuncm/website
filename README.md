# 🛍️ MaNar Store - AI-Powered E-Commerce

Welcome to **MaNar Store**! This is a full-stack e-commerce web application that I built from scratch using pure Vanilla JavaScript for the frontend and Node.js/Express for the backend. The coolest part about this shop is that it features an AI-powered semantic search assistant to help users find products easily.

I built this project as part of the Hack Club Stardance event!

---


##  Features I Implemented

* **Dynamic Product Grid:** Loads more than 55 products directly from the backend API. Users can sort and filter by category, price range, and ratings.
* **Smart AI Search:** An interactive chatbot assistant where you can type what you are looking for in plain English, and the backend AI returns actual product cards you can click and buy.
* **Persistent Cart:** A slide-out shopping cart drawer where you can change product quantities or remove items. It saves everything to `localStorage`, so your cart doesn't disappear if you refresh!
* **Full Authentication:** Working Sign Up and Sign In modals to manage user sessions.
* **Dark & Light Mode:** A clean toggle switch to change the website theme based on your preference.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, Vanilla JavaScript (No frameworks!)
* **Backend:** Node.js, Express.js
* **Data:** Managed via local state and variable-based persistence for seamless hosting.
