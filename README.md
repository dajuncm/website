# 🛍️ AI-Powered Online Shopping Platform

Welcome to the **AI-Powered Online Shopping Platform**! This is a modern e-commerce web application integrated with Artificial Intelligence to enhance the user shopping experience, provide smart recommendations, and automate customer interactions.

---

## 🚀 Features

* **Smart Product Search:** AI-driven search to help users find exactly what they are looking for.
* **AI Assistant / Chatbot:** Integrated AI chatbot to answer customer queries, help track orders, and suggest products.
* **Personalized Recommendations:** Dynamically analyzes user behavior to suggest relevant items.
* **Modern UI/UX:** A clean, responsive, and user-friendly interface for seamless shopping.
* **Cart & Checkout System:** Fully functional shopping cart with a smooth checkout flow.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript 
* **Backend:** Node.js, Express.js
* **AI Integration:** OpenAI API
* **Database:** users.db
